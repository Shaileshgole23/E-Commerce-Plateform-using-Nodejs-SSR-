# 🛒 E-Commerce Web Application

## 📖 About The Project
This is a full-stack E-Commerce web application built using **Node.js, Express.js, and MongoDB** with **Server-Side Rendering (SSR)** using EJS.  
The project delivers fast and dynamic content rendering while allowing users to browse and manage products efficiently.

---

## 🚀 Features
- Server-Side Rendering (SSR) using EJS  
- Product listing and detailed view  
- Add and manage products (CRUD operations)  
- MongoDB database integration  
- Responsive UI with Bootstrap  
- RESTful routing using Express.js  

---

## 🛠️ Tech Stack

**Frontend**
- HTML  
- CSS  
- Bootstrap  
- EJS  

**Backend**
- Node.js  
- Express.js  

**Database**
- MongoDB (Mongoose)
